Data Scrub

Project information

  • Category: Application
  • Project URL: Data-Scrub

Data-Scrub is a web application designed to allow users to upload CSV files and automatically sanitize sensitive or private information. The application identifies and removes or masks personally identifiable information (PII) such as names, emails, addresses, and phone numbers, providing a cleaned version of the data for secure use or sharing. Frontend: Developed with HTML, CSS, JavaScript, and Bootstrap to create a clean, responsive UI that allows users to easily upload CSV files, review the original data, and download the scrubbed output. Client-side validations and alerts are implemented for better user feedback. Backend: Built using Flask, the backend handles file uploads, processes CSV content, detects PII using pattern matching and basic heuristics, and returns a sanitized file. It includes error handling, secure file management, and customizable data-scrubbing logic. Key Features: Simple drag-and-drop or file selector interface for CSV uploads Automatic detection and scrubbing of PII Downloadable cleaned CSV file Clear display of original and sanitized data Lightweight and self-hostable with minimal dependencies
